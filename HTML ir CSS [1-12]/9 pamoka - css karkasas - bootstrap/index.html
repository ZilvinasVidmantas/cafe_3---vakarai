Bootstrap grid sistema

Bootstrap grid sistema, tai display: flex principu veikianti sistema. Ją sudaro eilutės ir stulpeliai.
eilutėms turime taikyti klasę .row ir jose kuriame stulpesiu su klasėmis kurios prasideda .col-*

Standartiškai stulpeliai bando išsiplėsti per visą eilutės plotį. Jeigu stulpelių eilutėje yra daugiau, tuomet jie dalinasi laisvą vietą lygiomis dalimis.Stulpeliams papildomai galime nurodyti plotį, pagal dvyliktainę sistemą:
col-1 - 1/12 tėvinio elemento pločio
col-2 - 2/12 tėvinio elemento pločio
col-3 - 3/12 tėvinio elemento pločio
col-4 - 4/12 tėvinio elemento pločio
col-5 - 5/12 tėvinio elemento pločio
col-6 - 6/12 tėvinio elemento pločio
col-7 - 7/12 tėvinio elemento pločio
col-8 - 8/12 tėvinio elemento pločio
col-9 - 9/12 tėvinio elemento pločio
col-10 - 10/12 tėvinio elemento pločio
col-11 - 11/12 tėvinio elemento pločio
col-12 - 12/12 tėvinio elemento pločio


Jeigu fiksuoto pločio stulpeliai netelpa į vieną eilutę, jie yra perkeliami į sekančią eilutę.

4 vienodo dyžio stulpeliai eilutėje

1 stulpelis užima pusę pločio, likusią dalį dalinasi 2 stulpeliai po lygiai

1 stulpeliai užima trečdalį, likusią dalį dalinasi 3 stulpeliai

Grid responsive

Bootstrap karkase standartiškai galioja tokie ekrano dydžiai:
xs - iki 576px
sm - nuo 576px
md - nuo 768px
lg - nuo 992px
xl - nuo 1200px
xxl - nuo 1400px


Šie dydžiai naudojami tam, kad nurodyti stiliaus savybių pakitimus NUO ekrano dydžio (sm, md, ...). Kuomet naudojame CSS klases, daroma prielaida, kad stilius rašote nuo 0px (nuo xs dydžio).

Nested grid

Naudojant Bootstrap grid sistemą, kartais yra aktualu turėti dalijamą eilutę stulpelyje. Kadangi eilutėje(.row) esantys stulpeliai (.col-*) užima plotį pagal tėvinio elemento plotį, grid sistema gali būti perpanaudota stulpelyje.
