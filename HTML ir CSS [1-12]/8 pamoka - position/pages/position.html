Position

Standarkiškai(default), jeigu nepasakome kitaip, visi elementai turi nustatytą CSS savybę position: static. Tai reiškia kad elementai yra nekintačios tėkmės (static flow). Static flow - tai elementų pozicionavimas iš kairės į dešinę, o kuomet elementai nebetelpa į tėvinio elemento plotį, perkeliami į sekančią eilutę.

Yra ir kitų posicionavimo tipų: relative, fixed, absolute ir sticky. Šie pozicionavimo tipai gali atsispirti nuo kitų elementų arba nuo jų nekintančios (static) vietos. Tai daroma naudojant savybes: top, right, bottom ir left. Static yra vienintelis pozicionavimo tipas, kuriam neturi įtakos savybės top, right, bottom ir left. Todėl jis ir yra vadinamis static.

Static

Static elementai nepriklauso nuo saybių top, right, bottom ir left ir yra atvaizduojami pagal nekintančią tėkmę(static flow).

Static flow - tai elementų pozicionavimas iš kairės į dešinę, o kuomet elementai nebetelpa į tėvinio elemento plotį, perkeliami į sekančią eilutę.
