Document Object Model

Document object model DOM tai medžio struktūros kintamasis, kuriame yra objektais suskaidytas visas puslapio turinys. DOM yra pasiekiamas kintamuosu document. Naudojant šį kintamajš galime surasti esančius elementus, ištrinti elementus, keisti jų klases, atributus ir pridėti naujus elementus.

Elementų paieškos funkcijos

document.querySelector(selektorius)

Tai funkcija skirta surasti pirmą HTML elementą pagal kriterijų. Standartiškai HTML elementų ieškome visame DOM'e, tačiau paieišką galime pradėti nuo bet kurio elemento.

Selektorius tai specialios sintaksės string'as skirtas apibūdinti elementui. Šio selektoriaus taisyklės yra identiškos kaip ir CSS failuose.
